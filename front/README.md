# Frontend Service
## Установка
1. Перейдите в директорию с фронтендом:
```bash
cd front
```
2. Убедитесь, что у вас установлен Node.js и npm.
3. Установите зависимости, выполнив команду:
```bash
npm install
```
## Запуск
1. Перейдите в директорию с фронтендом:
```bash
cd front
```
2. Запустите фронтенд сервис, используя команду:
```bash
npm start
```
